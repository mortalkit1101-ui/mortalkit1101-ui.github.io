Python 基础学习索引

发表于2026-07-15|更新于2026-07-15|Python
|浏览量:

学习目标

完成这一组笔记后，能够：

  • 使用变量、字符串和常见数据类型处理信息。
  • 根据需求选择列表、字典、集合或元组。
  • 使用判断、循环和函数组织程序。
  • 读写 TXT、JSON、CSV 文件并处理常见异常。
  • 独立解释并复现文本词频统计项目。

推荐顺序

  1. 变量、数据类型与字符串
  2. 列表、字典、集合与元组
  3. 条件判断、循环与函数
  4. 文件、JSON、CSV 与异常处理
  5. 文本词频统计项目实战

对应练习代码

掌握标准

  • 能解释 int、float、str、bool 的区别。
  • 能独立完成字符串切分、替换、清理和格式化。
  • 能用字典动态统计任意单词出现次数。
  • 能写出 if、for、while 和带返回值的函数。
  • 能用 Path 构造不依赖终端位置的文件路径。
  • 能读写 JSON 和 CSV，并处理文件不存在等异常。
  • 能解释 text-analyzer 中三个 Python 文件的职责。

建议练习方式

每学完一篇，先运行其中的示例，再关闭笔记独立重写。只有在不看答案时仍能写出，才算真正掌握。
