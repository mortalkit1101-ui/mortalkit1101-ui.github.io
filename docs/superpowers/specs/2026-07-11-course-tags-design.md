# 课程文章标签整理设计

## 目标

将课程文章的标签从知识点标签精简为课程归属标签，使标签页只承担课程检索作用，并在重新运行迁移脚本后保持一致。

## 标签映射

- 电源硬件与数字电源系列的 4 篇文章：统一为单一标签 `电力电子`。
- 微波工程与工程电磁场系列的 `02` 至 `08`：统一为单一标签 `电磁场`。
- 微波工程与工程电磁场系列的 `01`、`09` 至 `17`：统一为单一标签 `微波工程`。
- `00 微波工程与工程电磁场学习计划`：使用单一标签 `微波工程与工程电磁场学习计划`。

每篇文章不保留原有知识点标签，也不添加重复标签。分类和系列字段保持不变。

## 可维护性与验证

标签规则写入 `tools/apply-course-metadata.js`，迁移脚本重新导入文章后仍会自动应用。验证标准为：

1. 22 篇课程文章均只有一个标签。
2. 最终只生成 4 个标签页：`电力电子`、`电磁场`、`微波工程`、`微波工程与工程电磁场学习计划`。
3. 分类、系列、封面和文章顺序不受影响。
4. 本地构建及自动检查通过后，推送 GitHub Pages，并验证线上标签数量与代表性文章标签。
