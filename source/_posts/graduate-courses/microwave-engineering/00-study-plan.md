当前日期：2026-06-30  
目标期限：2026-07-31 前  
学习对象：

- 《工程电磁场导论 第2版》：约 436 页，8 章
- Pozar《微波工程》第四版：约 603 页，14 章

## 一、总目标

到 7 月底，不追求把两本书逐字精读完，而是达到：

- 能说清 Maxwell 方程、边界条件、平面波、反射折射、传输线、波导的物理意义。
- 能计算传输线输入阻抗、反射系数、驻波比、Smith 圆图匹配。
- 能读懂 S 参数、ABCD 矩阵、阻抗匹配、谐振器、功分器、滤波器等微波设计章节。
- 对放大器、振荡器、混频器、系统章节做到知道结构、指标和公式怎么用。

一句话：先补电磁场基础，再建立微波工程主线，最后按方向选读器件和系统。

## 二、学习顺序

不要两本书从第一页硬啃到最后。推荐顺序：

1. 用《工程电磁场导论》补电磁场主干。
2. 用 Pozar《微波工程》第 1-5 章建立微波工程核心框架。
3. 用 Pozar 第 6-8 章理解微波器件和网络设计。
4. 用 Pozar 第 10-14 章按方向选读。

## 四、每日学习方法

建议每天 4-5 小时。

1. 预习 30 分钟  
   看目录、小节标题和图，写下今天要解决的 3 个问题。

2. 正文阅读 90 分钟  
   只抓定义、公式来源、物理图像和适用条件。

3. 手推公式 60 分钟  
   关键公式自己推一遍，不求全，但要知道每一步从哪里来。

4. 例题和习题 90 分钟  
   每章挑 3-6 道代表题，不刷全量。

5. 闭书复述 30 分钟  
   写一页笔记：概念、公式、典型题、疑问。

每天结束问自己：

- 这个量的物理意义是什么？
- 这个公式适用条件是什么？
- 遇到题目先选哪条路径？
- 它和微波工程里的哪个东西相连？

## 六、习题策略

不要刷全书习题。时间不够，刷全量会拖垮主线。

每章选：

- 1 道概念题
- 2 道公式代入题
- 2 道综合计算题

重点多练：

- 平面波反射和透射
- 传输线输入阻抗
- Smith 圆图匹配
- S 参数矩阵判断互易、无耗、匹配
- 四分之一波长变换器
- Wilkinson 功分器基本分析
- 低通原型到带通、带阻的变换

## 七、笔记模板

每章笔记只写四类内容：

- 概念：这个章节解决什么问题。
- 公式：核心公式和适用条件。
- 题型：典型题的解题入口。
- 连接：它和前后章节的关系。

示例：

```text
传输线理论

核心对象：
V+、V-、Γ、Zin、VSWR

关键关系：
Γ = (ZL - Z0) / (ZL + Z0)
Zin = Z0 (ZL + jZ0 tanβl) / (Z0 + jZL tanβl)

物理图像：
负载不匹配 -> 反射 -> 驻波 -> 输入阻抗随位置变化

典型任务：
已知 ZL 和 l，求 Zin
已知 ZL，设计匹配网络
```

## 九、最终检查清单

到 7 月底，检查自己是否能完成这些任务：

- [ ] 写出 Maxwell 方程组，并解释每一条的物理意义。
- [ ] 解释电磁波为什么会传播。
- [ ] 算出平面波在介质分界面的反射和透射。
- [ ] 推出无耗传输线输入阻抗公式。
- [ ] 用 Smith 圆图完成单短截线匹配。
- [ ] 判断一个 S 参数矩阵是否互易、无耗、匹配。
- [ ] 解释矩形波导 TE10 模为什么重要。
- [ ] 解释 Q 值的物理意义。
- [ ] 说清 Wilkinson 功分器为什么能隔离。
- [ ] 说清低通原型如何变换成带通滤波器。
- [ ] 解释噪声系数、增益、动态范围的含义。
- [ ] 用 Friis 公式或雷达方程做一个基础系统估算。

## 十、一句话执行原则

7 月上旬补电磁场，中旬死磕传输线、Smith 圆图、S 参数和匹配，下旬学谐振器、功分器、滤波器，最后三天按方向选读有源电路或系统章节。

每天不要追求看了多少页，而要追求：

- 今天能不能闭书解释一个概念？
- 今天能不能推一个关键公式？
- 今天能不能做一道典型题？
