# 课程分类、系列与文章封面设计

## 目标

将博客中的研究生课程文章整理为两个符合 Hexo Butterfly 主题习惯的分类与连续阅读系列，同时删除测试文章，为每篇课程文章配置合适的卡片封面，并在推送 GitHub 后确认线上博客可正常展示。

## 内容范围

- 删除 `source/_posts/welcome-to-mortal.md`，不再生成 “Welcome to Mortal”。
- 保留现有 22 篇课程文章正文，不补全课程知识，也不上传 PDF。
- 电源课程包含 4 篇文章；微波课程包含 18 篇文章。
- 不重新创建此前删除的 `/courses/` 资料链接页。

## 分类入口

首页右侧使用 Butterfly 原生“分类”卡片作为合集入口。分类卡仅展示：

1. 电源硬件与数字电源
2. 微波工程与工程电磁场

每篇文章只归入上述两个课程分类之一，移除会导致侧栏出现额外分类的父级“研究生课程”分类。点击分类名称进入 Butterfly 原生分类归档页，该页面展示对应课程的全部文章。

## 系列与顺序

两门课程同时使用 Butterfly 原生 `series` 元数据：

- `电源硬件与数字电源`
- `微波工程与工程电磁场`

系列按文章标题升序排列。文章标题统一以两位编号开头，确保字典序与学习顺序一致：

- 电源：`00` 学习路线、`01` 电力电子绪论、`02` 功率二极管、`03` 双极结型晶体管 BJT。
- 微波：沿用 `00` 至 `17` 的现有编号。

文章页显示当前系列目录，并保留 Butterfly 的上一篇/下一篇导航，形成连续阅读路径。分类归档页也按标题编号升序展示，而不是按发布日期排序。

## 文章卡片封面

所有 22 篇课程文章必须包含有效的 `cover` 路径，首页和分类归档页不得出现无图白框。

- 已有内容相关且清晰的电源文章图片继续作为封面。
- 电源学习路线与微波课程文章补充独立封面。
- 封面采用统一的技术插画语言与横向比例，但每篇根据主题使用不同的核心图形，例如电路、半导体结构、电场线、波形、传输线、Smith 圆图、波导、谐振器、耦合器和滤波器。
- 图片中不放细小正文和教材名称，避免卡片裁剪后无法辨认。
- 新增封面统一存放在 `source/img/courses/covers/` 下，并使用稳定、可读的英文文件名。

## 可维护性

更新 `tools/migrate-graduate-courses.ps1` 中的文章映射，使重新运行迁移脚本后仍能得到相同的：

- 两个课程分类；
- 系列名称；
- 两位数文章编号；
- 封面路径；
- 文章页系列返回与连续阅读结构。

不修改原始 Obsidian 资料目录。

## 构建与验证

本地验收包括：

1. 执行 Hexo 清理和生成命令并成功退出。
2. 确认生成结果不含 Welcome 文章与旧 `/courses/` 合集链接页。
3. 确认首页分类卡只出现两个课程分类。
4. 确认两个分类页分别包含 4 篇和 18 篇文章，并按编号升序排列。
5. 确认 22 篇文章均存在可加载的卡片封面。
6. 抽查两门课程的首篇、中间篇和末篇，确认系列目录与上一篇/下一篇链接正确。
7. 检查生成目录中不存在被引用但缺失的本地图片。

## GitHub 发布

本地验收通过后提交博客源码并推送到用户的 GitHub 仓库。随后检查 GitHub Actions / Pages 部署状态，并访问线上博客验证：

- 首页正常加载；
- 分类卡与两个分类页正常；
- 文章排序、封面和系列导航正常；
- 页面无明显 404 图片或样式缺失。

仅在源码推送、部署成功和线上页面检查均完成后，视为任务完成。
