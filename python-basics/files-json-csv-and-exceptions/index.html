1. 使用 Path 构造路径

  • __file__：当前 Python 文件。
  • resolve()：得到绝对路径。
  • parent：得到所在文件夹。
  • /：拼接路径。

这种写法不依赖终端当前目录。

3. JSON

JSON 适合保存字典、列表等结构化数据。

  • ensure_ascii=False：保留中文。
  • indent=2：使用两个空格缩进。
  • json.dump()：写入文件。
  • json.load()：从文件读取。

常见错误

  • 相对路径错误：程序寻找的位置取决于终端当前目录。
  • 输出文件夹不存在：写文件前用 path.parent.mkdir(parents=True, exist_ok=True)。
  • 忘记编码：中文文本应显式指定 encoding="utf-8"。
  • w 模式误覆盖：需要保留旧内容时使用 a。
