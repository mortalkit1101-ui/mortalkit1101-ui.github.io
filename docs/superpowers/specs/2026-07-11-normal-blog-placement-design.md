# 正常博客文章布局调整设计

## 目标

参考 `https://blog.linkwushuang.top` 的文章组织方式，而非复制其 Hugo Reimu 视觉主题。研究生课程笔记应作为普通博客文章出现在首页、分类、标签、归档和搜索中，不再通过原始资源链接页展示。

## 调整范围

- 保留 Hexo Butterfly 主题与当前个人配色、背景、头像和侧栏。
- 保留已经生成的 22 篇标准 Hexo 课程文章及其图片、公式和固定链接。
- 从顶部导航移除 `Courses` 菜单。
- 删除 `source/courses/index.md` 课程目录页，使 `/courses/` 不再作为链接集合入口。
- 首页继续使用 Butterfly 文章卡片，展示封面、标题、日期、摘要和分类；课程文章与普通文章遵循同一展示方式。
- 使用“电源硬件与数字电源”和“微波工程与电磁场”两个课程分类组织内容，用户从首页、分类、标签、归档和搜索进入文章。
- 文章底部删除“返回课程目录”，保留课程内上一篇与下一篇链接；课程首尾只显示存在的相邻文章。
- 不恢复任何 PDF、原始 Markdown、文件大小或附件下载列表。

## 内容流

```text
首页文章卡片
  → 课程文章正文
  → 上一篇 / 下一篇

分类页
  → 电源硬件与数字电源
  → 微波工程与电磁场

归档 / 标签 / 搜索
  → 对应课程文章
```

## 验收

1. 顶部导航不再出现 `Courses`。
2. `/courses/` 不再生成课程链接列表。
3. 首页文章总数为 23，其中课程文章为 22。
4. 两个课程分类分别显示 4 篇和 18 篇文章。
5. BJT 文章正文、7 张图片和 MathJax 公式正常显示。
6. 文章内没有 `/courses/` 目录返回链接，也没有失效的上一篇/下一篇。
7. Hexo 清理和生成成功，源码与生成目录均无 PDF、Obsidian 内链或原始附件列表。
8. 本地浏览器检查桌面端和移动端首页、分类页与 BJT 文章。

## 非目标

- 不切换到 Reimu 主题。
- 不复制参考博客的动漫图片、颜色、字体、侧栏组件或文章内容。
- 不改变现有文章正文的知识范围。
