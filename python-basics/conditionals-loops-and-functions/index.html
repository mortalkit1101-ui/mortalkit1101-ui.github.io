常见错误

  • 缩进错误：Python 使用缩进表示代码块，通常使用 4 个空格。
  • while 条件永远为真：忘记更新变量会形成死循环。
  • 文件没有保存就运行：导入的模块可能仍是磁盘上的空文件。
  • 函数只打印但没有 return：调用结果会是 None。
