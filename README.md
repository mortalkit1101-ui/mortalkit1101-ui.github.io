# Motral

A static Liquid Glass personal blog interface inspired by Navfolio-style personal site structure.

## Files

- `index.html`: the whole static interface
- `styles.css`: liquid glass visual system
- `script.js`: contribution grid and small interactions
- `assets/sky-meadow.png`: generated sky meadow background

## Deploy

This project is static. You can deploy the `Blog` folder directly to GitHub Pages.

For a project page, push these files to a GitHub repository and enable Pages from the repository settings.
