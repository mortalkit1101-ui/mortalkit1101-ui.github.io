# 电力电子课程封面统一设计

## 目标

将电力电子课程 01–03 文章的首页封面从正文 PNG 改为与微波课程相同的 SVG 视觉系统，同时保留课程辨识度。

## 已确认方案

采用方案 B：同版式、不同学科色。

- 画布统一为 1200×675，圆角 36。
- 沿用微波封面的渐变背景、42 px 网格、右侧柔光、章节号、中文标题、英文副标题、分隔线与底部装饰。
- 电力电子统一使用深蓝到青色渐变；微波课程继续使用紫色。
- 右侧使用白色线性技术图标，不直接使用正文截图。
- 00 学习路线已有同类 SVG，保持现状。

## 封面映射

| 文章 | 新封面 | 右侧图标 |
|---|---|---|
| 01 电力电子绪论 | `power-01-introduction.svg` | 能量流/开关变换方框与箭头 |
| 02 功率二极管 | `power-02-power-diode.svg` | 标准二极管符号 |
| 03 双极结型晶体管 BJT | `power-03-bjt.svg` | NPN BJT 线性符号 |

## 内容引用

只修改三篇文章 Front Matter 的 `cover` 字段。正文中的原始 PNG 和讲解图片全部保留，不改变正文内容。

## 验证

1. 三个 SVG 均能被浏览器加载，尺寸和 `viewBox` 一致。
2. Hexo 完整生成成功。
3. 首页三张卡片显示新封面，标题不会与右侧图标重叠。
4. 分类页、标签页和文章页封面资源无 404。
5. 站点校验更新为允许新增“规划”分类与“研究生期间规划”标签，并通过。

## 范围外

- 不修改正文插图。
- 不改变微波课程封面。
- 不引入 PNG/JPEG 位图封面或新的图片生成依赖。
